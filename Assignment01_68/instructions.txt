1. Overview 
This assignment is important to prepare the organization of the ideas that you might have 
for the Midterm Project.  
You will download the Examples from the Bootstrap web page and will select the template 
of  your  preference.  For  your  selection  you  must  consider  the  example  that  fits  better 
according to your idea of the Midterm Project. 
 
This assignment has been focused on the main idea of “Catalog of Products” shown in 
class the Friday January 2nd, the Catalog of Flowers and the instructions are close related 
as if were developing a catalog of a product. However, you can select a different example 
from Bootstrap to develop your Assignment 01. 
 
You will collect images and text from Internet or created from your original creativity, such 
as  pictures  using  your  cellular  phone  or  any camera. And the text  can  be  developed  by 
yourself according to the images collected or created. 
 
This  assignment  will  be  developed  by  the  student  members  of  the  Team  formed  for  the 
Midterm project. 
  
The case shown in class Activity 05 Catalog of flowers can be taken as the main idea to 
develop this assignment.  
 
2. You will create a Web site with the next characteristics : 
• The web site will contain at least 2 web pages. 
• Use Bootstrap CSS style. 
• At least a set of 5 images and its corresponding text 
(The text is in free format and can be compose of one or several sentences)   
• Implement a header of your preference. 
• Implement a footer of your preference. 
• Implement a top or left Navigation bar. 
The navigation bar will allow you to visit the different web pages of the Web site. 
The navigation bar can optionally allow you to visit one page located on the Internet. 
• One page with the credits, the complete information of the student members. 
• The index.html is the root file of the Web site.  
• There will be a folder of images with name images where all of them are contained. 
• The different text per image are embedded in the html files.  
 
 
 
The name of the principal Web page will be index.html and the name of secondary web 
pages will be according to the preferences of the students. 
This assignment if composed only of html and images (png, jpg, etc).
